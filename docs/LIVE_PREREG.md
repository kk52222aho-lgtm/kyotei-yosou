判定は「暦」でなく「ベット本数」ゲートで（ライブの賭け頻度は不確実だから）。低分散の裾の軽い順に読む。

## 生死ライン（固定）

### 単勝≥1.5
- **N<300本: 判定しない**
- N≥300本で:
  - **畳む**: ROI < 100%（控除に負け＝エッジ消滅）
  - **信じる**: ROI ≥ 108%（机上116%と方向一致・黒字）
  - 100-108%: 継続・保留

### Track A 2連単（無フィルタ土台）
- **N<600本: 判定しない**（fat-tail）
- N≥600本で:
  - **畳む**: ROI < 110% かつ 上位5%的中を除いても < 100%
  - **信じる**: ROI ≥ 150% かつ 上位5%除外でも ≥ 120%
  - 中間: 継続

### ③ EV>2.0 2連単（最重fat-tail・派手だが当分読めん）
- **N<300本(~1年+): 一切更新しない。** 爆下げも爆上げも無視する（万舟ゼロ/1発で±両方起きるから、どっちも真の平均でない）
- N≥300本で:
  - **畳む**: ROI < 100%（控除割れ＝エッジ来なかった）
  - **信じる**: ROI ≥ 200% **かつ 上位5本的中を除いても ≥ 130%**（1-2発の万舟牽引でないことを必須確認）
  - 100-200%: 継続・未決
