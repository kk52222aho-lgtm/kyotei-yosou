## 🚧 利益ライン（線引きの核心）— 単勝フラットの話（上の「勝てる形」が突破口）

```
   回収率 100%（控除率の壁／損益分岐）
   ─────────────────────────────────────────
        ↑ ここを安定的に超えたものは「まだ無い」
   99.2%  本命≠1号艇（大サンプル）   ← "フラットよりマシ" 止まり
   ~93%   本命≠1号艇（別期間）
   ~92%   単勝フラット全体           ← 構造的に負ける（確定）
   ~91%   本命=1号艇                 ← 人気イン素張りは控除率に負ける
```

- **現時点、どの賭式・条件も回収率100%を安定的に超えない。**
- 「本命≠1号艇」は **"フラットより損が小さい"(93〜99%)** であって **"儲かる"(>100%)ではない**。← これが線。
- **結論: 現状のモデル＋単勝フラットは「当たる予想」としては成功、「勝てる馬券」としては未到達。**
